Описание AI Chat проекта
Это десктопное приложение для работы с AI чатами на Python с поддержкой множества провайдеров и RAG функциональности.

Архитектура
launcher.py - точка входа, скрывает консоль, управляет запуском
ui.py - основной GUI на CustomTkinter (~32k токенов)
cross_gpt.py - ядро системы, глобальное состояние, интеграция с AI провайдерами
chat_manager.py - управление чатами (стандартный и RAG режимы)
rag_constructor.py - векторная база ChromaDB, семантический поиск
AI провайдеры
Поддержка: OpenAI, Anthropic, Cohere, Ollama, LM Studio, GPT4All, HuggingFace Hub

RAG функциональность
Векторизация сообщений в ChromaDB
Семантический поиск релевантного контекста
Иерархические резюме для управления контекстом
Контекстно-зависимые ответы
Модули
system_tools/librarian.py - получение недостающей информации
default_tools/ - стандартные инструменты (файлы, веб-поиск, команды ОС)
model_providers/ - интеграция с различными AI API
lang/ - локализация (ru/en), 200+ системных текстов
Особенности
Многопроцессная архитектура
Кэширование результатов (SQLite)
Система логирования
Поддержка вложений и изображений
OCR (EasyOCR) и анализ изображений (BLIP)
Каждый чат хранится в отдельной папке в data/chats/ с настройками, кэшем и RAG базой.
